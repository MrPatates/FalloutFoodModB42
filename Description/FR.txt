[h1]Ce mod est un portage du [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2843941815] mod Fallout Food [/url] vers la B42.[/h1]

[h2]Ce mod ajoute des aliments d'avant-guerre de l'univers Fallout[/h2]

Lorsque vous ouvrez une Nuka Cola, vous recevrez une capsule de bouteille, qui pourrait être utile pour les échanges.
Les objets peuvent être trouvés aussi bien dans les magasins que dans les maisons.
